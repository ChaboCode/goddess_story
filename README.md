# Goddess Story Plugin

### Manage your Goddess Story boxes 
![Odoo E-Commerce products listing, grouped by E-Commerce categories](/pics/list.png)
- Atomatically create inventory items for **boxes** and **packs**
- Manage inventory for **sealed** and **open** boxes for individual pack selling
- Includes e-commerce plugin category for an easier online selling experience
- Simple interface: add your new sets with ease
![Menu for adding a new Godess Story set](/pics/new_menu.png)
